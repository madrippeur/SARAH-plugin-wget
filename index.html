Simple plugin qui permet d'executer une requête http sur un serveur web distant.
Remplir le fichier wget.prop en lui donnant l'adresse jusqu'au dossier contenant les fichiers a faire executer.

Par exemple sur le serveur web, creer un dossier "sarah" dans lequel on place le fichier montruc.php.
Evitez les noms avec des accents et des espaces pour eviter tout bug et donnez ette adresse au wget.prop :
http://mondomaine.com/sarah

Ensuite remplissez le fichier wget.xml en lui rajoutant une ligne de type :

'MA COMMANDE VOCALE out.action.action="fichier"; out.action.fichier="MONFICHIER.EXTENTION"; '
